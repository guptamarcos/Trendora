🛍️ Trendora – Modern MERN E-commerce Platform

Trendora is a full-stack e-commerce web application built using the MERN stack (MongoDB, Express, React, Node.js). It provides a seamless shopping experience with modern UI, secure authentication, and scalable backend architecture.

🚀 Features
🧑‍💻 User Features
🔐 Authentication (JWT-based login/signup)
🛒 Add to Cart & Persistent Cart
🔍 Product Search & Filtering
📦 Product Details Page
❤️ Wishlist functionality

🛠️ Admin Features
👤 User Management
📦 Product Management (CRUD)
🏷️ Category & Collection Management
💰 Discount System
📊 Order Management

⚡ Advanced Features
🔄 Real-time notifications
☁️ Image uploads (Cloudinary)

🧱 Tech Stack
Frontend
React.js
Context Api (State Management)

Express.js

Authentication
